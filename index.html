Comandos Git

Inicialização das atividades

  • git clone "link repositório git"

    Copia os arquivos do repositório remoto para o local

  • git fetch --all

    Sincroniza o repositório local com repositorio remoto

  • git pull

    Faz download das atualizações do servidor remoto

Finalização das atividades

  • git status

    Mostra oque tem de diferente no servidor

  • git add .

    adiciona as modificações do servidor local

  • git commit -m "texto"

    Adiciona uma comit no github com o texto que você quiser

  • git push

    Faz o upload dos novos arquivos para o servidor remoto
